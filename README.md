````md
## 프로젝트 개요

이 프로젝트는 초등학생이 부담 없이 구구단을 반복 학습할 수 있도록 만든 학습용 웹앱입니다.

핵심 목표는 아래와 같습니다.

- 단순하고 쉬운 흐름
- 즉시 정오답 피드백
- 오답 복습 가능
- 모바일에서도 누르기 쉬운 UI
- 밝고 예쁜 학습 분위기

---

## 주요 기능

- 시작 화면에서 `2단` ~ `9단`, `전체 섞기` 중 하나를 선택해 퀴즈 시작
- 한 세션당 10문제 출제
- 답을 누르면 즉시 정오답 피드백 표시
- 동일 문제에서 중복 채점 방지
- 결과 화면에서 정답 수, 오답 수, 정답률 확인
- 오답이 있으면 오답만 다시 풀기 가능
- `/all-dan` 페이지에서 전체 구구단 보기 및 직접 연습 가능
- 모바일 폭에서도 버튼 터치 영역을 고려한 UI

## 실행 방법

패키지 매니저는 `pnpm` 기준입니다.

### 1) 의존성 설치

```bash
pnpm install
````

### 2) 개발 서버 실행

```bash

```

개발 서버 실행 후 아래 주소에서 확인할 수 있습니다.

```text
http://localhost:3000
```

## 화면 구성

### `/`

메인 퀴즈 페이지입니다.

포함 화면:

* 시작 화면: 연습할 단 선택 후 시작
* 퀴즈 화면: 진행률, 점수, 문제, 선택지, 즉시 피드백 제공
* 결과 화면: 점수 요약, 오답 목록, 오답 다시 풀기 제공

### `/all-dan`

전체 구구단을 한 번에 보고 직접 눌러서 연습하는 페이지입니다.

포함 기능:

* 2단부터 9단까지 카드형 전체 보기
* 단과 곱하는 수를 고른 뒤 정답 토글로 직접 연습
* 원하는 단 카드를 확대해서 집중해서 보기 가능

---

## 동작 로직 요약

* 일반 퀴즈는 선택한 단 기준으로 문제를 생성합니다.
* `전체 섞기`는 2단부터 9단 사이에서 문제를 랜덤하게 섞어 출제합니다.
* 각 문제는 보기 4개로 구성됩니다.
* 답을 선택하면 즉시 정답 여부를 판정합니다.
* 한 문제는 한 번만 채점됩니다.
* 오답 다시 풀기는 결과 화면에 쌓인 오답 목록만 새 문제 세트로 재구성합니다.
